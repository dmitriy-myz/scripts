# WebTail

Simple tail for http-served content. E.g. logs.  
Written on bash

## Requirements
1. bash
2. grep
3. curl
4. awk

## Installation
copy webtail.sh to any directory.  
If you want to install globaly copy to directory in you'r PATH. E.g. `/usr/bin`


## Usage
```bash
webtail.sh $url [bytes]
```
